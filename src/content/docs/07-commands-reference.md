## 初始化与立项 (Day 0)

1. **`user`**
   **[描述]**：建立你的基本人设、个人职业背景与核心初始短板。
   **[隐性影响]**：不同的年龄体能及职业经历会彻底固化你的发展长板与初始创业资本，是定生死的开局指令。
   
2. **`corp`**
   **[描述]**：建立公司外壳，选定你要搏杀的残酷赛道（行业）和你想如何收割财富（商业模式）。
   **[隐性影响]**：系统将围绕你的选择，搭建针对性的行业考核难度与市场偏移度。

3. **`idea`**
   **[描述]**：用一句话向系统提交你的构思。
   **[隐性影响]**：AI 评审组将无情地剖析此 Idea，这将为你第一款产品的成熟度垫基。这同样也会用在未来多线并发时的立项评审。

## 核心经营循环 (The Core Loop)

4. **`dev`**
   **[描述]**：推进研发，让进度条向前滚动（消耗巨量精力与现金流）。
   **[隐性影响]**：最核心的“消耗进度”利器，但长期死磕而缺乏规划必然导致技术瘫痪式的反噬。

5. **`fix`**
   **[描述]**：暂停进取，用来修复团队混乱或系统技术债。
   **[隐性影响]**：用来喘息的防暴毙技能。它能有效降低指数级的内耗熵值，让公司重回高潜轨道，代价则是坐看机会流转。

6. **`scan`**
   **[描述]**：向外窥探。扫描当下市场动向与团队脱节度分析。
   **[隐性影响]**：能为你揭示市场正在悄然朝哪个死亡象限偏航，并给予及时的重组警告。

7. **`test`**
   **[描述]**：产品投放，进行 A/B 测试或者尝试从用户那里收取第一笔报酬。
   **[隐性影响]**：唯一可以将代码产品转化为现金变现与市场反馈的通道。

8. **`prod`**
   **[描述]**：对公司所持有的多条产品线进行重组、焦点切换与斩断止损。
   **[隐性影响]**：多线作战由于组织焦点分散带来的“超爆内耗”只能依靠 `prod` 主动挥刀自宫，关停旁支或者彻底 Pivot。

## 组织扩张与中后期博弈 (Scaling)

9. **`hire`**
   **[描述]**：扩编人类员工或购买自动化代理（AI Agent）来填补你的能力短板。
   **[隐性影响]**：人多不等于能力强，员工有配合度与摩擦损耗。找错人，产出可能直接开倒车。

10. **`auto`**
    **[描述]**：用技术建立自动化防线（CI/CD、自动化客服等 workflow）。
    **[隐性影响]**：能够一劳永逸地将某一些低级操作成本锁死，但在极度红海中需要付出沉没的研发验证成本。

11. **`grow`**
    **[描述]**：重资源打法：广告轰炸、媒体 PR 大推。
    **[隐性影响]**：极其吞噬金钱（需要极高 MKT 才能见效），玩不好就是白撒钱，玩得好能带来指数级的月度收入狂破。

12. **`fund`**
    **[描述]**：对资方发起募资宣讲路演。
    **[隐性影响]**：出让股权换回续命神仙水。高难度下你将体会到投资人的苛刻条款与短视要求。

13. **`buy`**
    **[描述]**：发起并购申请（M&A），只适用于超高估值晚期。
    **[隐性影响]**：直接铲除那些可能在未来击穿你天花板的高能效竞品，稳固统治霸权。

14. **`rule`**
    **[描述]**：影响监管，建立排他式行标。
    **[隐性影响]**：登峰造极的垄断玩法，用以维持在极红海中的不败之境，成功概率极低。
